HOW TO USE
================================================================================

1. MOUSE WHEEL SCROLLING
   Position mouse over terminal → Scroll wheel ↑↓

2. SCROLLBAR THUMB DRAGGING
   Click on blue scrollbar → Drag up/down to navigate

3. SCROLLBAR TRACK CLICKING
   Click above scrollbar thumb → Page Up
   Click below scrollbar thumb → Page Down

4. KEYBOARD SHORTCUTS
   Page Up    → Scroll up one page
   Page Down  → Scroll down one page
   Home       → Jump to top of terminal
   End        → Jump to bottom of terminal
