# Cortex AI Agent - Title Generation Prompt
# Use this to generate concise, descriptive chat titles

Generate a short, descriptive title (3-6 words) for this conversation based on the user's first message.

## RULES

- Maximum 6 words
- Capture the main intent
- Use action verbs when appropriate
- Be specific, not generic
- No punctuation at the end

## EXAMPLES

User: "Create a login system with JWT authentication"
Title: "JWT Authentication Implementation"

User: "How does the React useEffect hook work?"

User: "Fix the bug where users can't logout"
Title: "Logout Bug Fix"

User: "Refactor the database connection code"
Title: "Database Connection Refactor"

User: "What files are in the src directory?"
Title: "Source Directory Exploration"

## OUTPUT FORMAT

Respond with ONLY the title text. No quotes, no explanation, no extra text.
